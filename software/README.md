# Software

Aplicativo desktop da BRSWDIY para Windows.

## Objetivo do MVP

- detectar portas seriais
- conectar à base
- executar handshake
- ler status e configuração
- ajustar ganho global e limite de saída
- exibir respostas e erros de forma legível

## Stack inicial

- Python 3.11+
- pyserial
- DearPyGui
- pytest
